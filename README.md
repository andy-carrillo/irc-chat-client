# IRC Chat Client (Python)

Un cliente IRC ligero escrito en Python, inspirado en XChat y diseñado para conectarse a servidores como `irc.irc-hispano.org`.

## 🚀 Requisitos

- Python 3.6 o superior
- Terminal o Visual Studio Code con extensión Python

## 🧱 Estructura básica

- `irc_client.py`: Script principal para conectarse y chatear por IRC
- `.gitignore`: (opcional) Para ignorar archivos temporales
